# 海外預金 税務シミュレーター（日本居住者向け）

海外の銀行に預金を持つ日本居住者向けに、利息にかかる現地の源泉税と、日本での確定申告の要否・外国税額控除を概算するWebアプリです。

**公開URL: https://jp-overseas-deposit-tax.vercel.app**

## 背景

海外預金の利息は、日本の国内預金（20.315%の源泉分離課税）とは扱いが異なり、総合課税として累進税率で課税されます。さらに現地で源泉徴収された税は外国税額控除の対象になりますが、控除には限度額があり、租税条約の有無によって適用税率も変わります。

こうした判断は税理士でも手間のかかる領域ですが、多くの預金者は「まず自分に申告が必要なのか」「手取りはいくらになるのか」という目安を知りたいだけです。本アプリはその概算を、最小限の入力で即座に提示します。

## 機能

- **第1段階**：預入国・預金額・年利・期間・為替レートから、利息と現地源泉税を試算し、確定申告の要否を判定
- **第2段階**：申告不要と判定された場合、どのような条件で課税・申告が発生するかを解説
- **第3段階**：申告が必要な場合、日本の累進課税と外国税額控除の限度額を計算し、最終手取りと実効税率を表示

## 実装上の工夫

- **国別税率マスタの分離**：16か国の源泉税率・租税条約の有無をデータとして本体ロジックから切り離し、国の追加や税率更新を容易に。リストにない国は手入力にも対応
- **累進課税の差額計算**：利子を「含む場合」と「含まない場合」の所得税を両方計算し、その差分を利子にかかる税とすることで、累進税率の積み上がりを正確に反映
- **外国税額控除の限度額**：所得税・復興特別所得税・住民税の枠（所得税限度額×1.321）で概算し、控除しきれない分は3年繰越の対象として明示
- **入力バリデーション**：必須項目の明示と、未入力時に誤った判定を出さないガード
- **レスポンシブ対応**：PC・スマートフォンの両方で自然に読める組版
