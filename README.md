# 海外預金 税務シミュレーター（日本居住者向け）

海外の銀行に預金を持つ日本居住者向けに、利息にかかる現地の源泉税と、日本での確定申告の要否・外国税額控除を概算するWebアプリです。日英バイリンガル対応。

**公開URL: https://jp-overseas-deposit-tax.vercel.app**

## 背景

海外預金の利息は、日本の国内預金（20.315%の源泉分離課税）とは扱いが異なり、総合課税として累進税率で課税されます。さらに現地で源泉徴収された税は外国税額控除の対象になりますが、控除には限度額があり、租税条約の有無によって適用税率も変わります。

こうした判断は税理士でも手間のかかる領域ですが、多くの預金者は「まず自分に申告が必要なのか」「手取りはいくらになるのか」という目安を知りたいだけです。本アプリはその概算を、最小限の入力で即座に提示します。

## 機能

- **第1段階**：預入国・預金額・年利・期間・為替レートから、利息と現地源泉税を試算し、確定申告の要否を判定
- **第2段階**：申告不要と判定された場合、どのような条件で課税・申告が発生するかを解説
- **第3段階**：申告が必要な場合、日本の累進課税と外国税額控除の限度額を計算し、最終手取りと実効税率を表示
- 単利（満期一括）と複利（毎年受取・再投資）に対応。年ごとの内訳を表示
- 円転時の為替差益を任意で計算（雑所得として判定に反映）
- 結果の印刷 / PDF保存、入力条件をURLで共有
- 日本語 / English の切替

## 実装上の工夫

- **国別税率マスタの分離**：26か国の源泉税率・租税条約の有無をデータとして本体ロジックから切り離し、国の追加や税率更新を容易に。リストにない国は手入力にも対応
- **累進課税の差額計算**：利子を「含む場合」と「含まない場合」の所得税を両方計算し、その差分を利子にかかる税とすることで、累進税率の積み上がりを正確に反映
- **課税タイミングの区別**：満期一括なら満期年に一括課税、複利なら毎年課税。申告要否の判定に用いる金額が変わるため、受取方法によって計算基礎を切り替える
- **為替差益の正しい取り扱い**：雑所得として総所得に加算する一方、国外源泉所得ではないため外国税額控除の限度額の分子には含めない。外貨のまま保有する元本には差益を認識しない（所得税法施行令167条の6第2項）
- **外国税額控除の限度額**：所得税・復興特別所得税・住民税の枠で概算し、控除しきれない分は3年繰越の対象として明示
- **出典の明示**：財務省・国税庁・ジェトロの一次情報へリンクし、税率が代表値であること、条約の限度税率を超える源泉徴収分は控除対象外となりうることを注記
- **スコープを明示した多言語化**：UIは日英対応だが、税務エンジンは日本の税制のみを実装している。英語版には「日本の居住者にのみ有効」というスコープ注記を表示し、他国のユーザーの誤解を防ぐ
- **入力バリデーション**：必須項目の明示と、未入力時に誤った判定を出さないガード
- **サンドボックス耐性**：`file://` やiframe内でも動作するようブラウザAPIの失敗を安全に処理
